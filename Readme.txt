code is compiled with premake4.
"premake4.lua" contains the specifics for building the project.
Libraries used are listed in "header.hpp"

"main.cpp" corresponds to the main file of the code.
"class" folder contains the different classes defined in this code.
"input.conf" contains the variable input list.
"plotGaps.m" process the output and extracts data for size, life time and frequency of gaps.
